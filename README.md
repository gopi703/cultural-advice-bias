# Cultural Advice/Therapy Bias Visualization

An interactive visualization tool that analyzes and displays epistemic erasure and cultural bias in AI-generated advice/therapy recommendations across different cultural backgrounds (Filipino, Indian, and Nigerian contexts).

## ğŸ¯ Project Overview

This project examines how AI systems handle culturally specific mental health scenarios. It reveals patterns of **epistemic violence/Erasure**  when AI replaces local knowledge systems with dominant Anglophone psychological frameworks, effectively erasing culturally-grounded approaches to wellbeing.

### Key Research Questions
- How accurately does AI address culture-specific dilemmas?
- What geographic sources does AI rely on for therapeutic advice?
- Which psychological frameworks dominate AI's responses across different cultural contexts?
- How does AI handle concepts that don't translate neatly into Anglophone Psychology?


## ğŸ” Key Findings

Across 101 AI-cited sources for Indian, Nigerian, and Filipino users: **40-44% of sources come from the US** despite culturally-specific questions. **Only 12-14% directly address the user's concern**, while **Filipino context showed highest generic advice rate 62%** and **strongest boundaries** emphasis. Top keywords across all contexts: "boundaries," "budget," "financial independence" this reflects individualist Anglophone frameworks imposed on collectivist cultures. AI retrieves what's most accessible (digitized, English, SEO-optimized), but this accessibility isn't neutral. it systematically privileges Anglophone psychological discourse over local knowledge.

## ğŸ”¬ Methodology

### Three-Phase Analysis

**Phase 1: Multi-turn Conversations**
- Three users (Filipino, Indian, Nigerian) engage AI in natural 4-5 turn conversations
- Each discusses culturally-specific dilemmas (family obligations, marriage customs, financial arrangements)
- AI provides advice and cites sources to support recommendations

**Phase 2: Source Analysis**
- Extract and categorize all URLs cited by AI
- Analyze geographic origin of sources
- Identify psychological frameworks present (boundaries, individualism, financial independence etc)
- Assess cultural relevance: Does the source address the user's specific concern or provide generic advice?

**Phase 3: Pattern Recognition**
- Map Anglophone psychological keywords appearing across cultural contexts
- Identify when Global North frameworks are applied to Global South situations
- Detect epistemic erasure: when local knowledge is replaced by dominant frameworks

### What We Measure

**Geographic Bias**
- Where do AI's cited sources originate?
- How much advice comes from the user's own region vs. Anglophone publishers?

**Framework Imposition**
- Which psychological concepts appear most frequently? (boundaries, self-care, individualism)
- Are these frameworks appropriate for collectivist cultural contexts?

**Relevance Assessment**
Categories for each cited source:
- â­**Addresses user's concern** - Directly relevant to the specific dilemma
- ğŸ“š **Explains concepts** - Educational, defines cultural practices
- ğŸŒ **Generic advice** - General guidance, not culturally specific
- ğŸ“ **Far from User's Concern** - Tangentially related at best
